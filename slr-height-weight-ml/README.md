# Height vs Weight Prediction (Simple Linear Regression)

This project predicts human height based on weight using **Simple Linear Regression**.

## Dataset
- Input: Weight (Pounds)
- Output: Height (Inches)

## Workflow
- Data visualization
- Train-test split
- Feature scaling
- Model training
- Evaluation (MAE, MSE, RMSE, R²)
- Residual analysis
- Prediction on new data

## Model Used
- Linear Regression (Scikit-learn)

## Results
- R² Score shows how well weight explains height variation

pip install -r requirements.txt
